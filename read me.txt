bcs203077-Currency Converter
This is a Python program that allows you to convert currency from one unit to another. It uses the exchange rates API to get the latest exchange rates.

Prerequisites
Before running the program, make sure that you have the following:

Python 3.6 or higher installed on your computer
The requests library installed. If you don't have it, you can install it by running pip install requests in your terminal/command prompt.
Running the Program
To run the program, follow these steps:

Download or clone the repository to your computer.
Open your terminal/command prompt and navigate to the folder where the program is located.
Run the command python currency_converter.py.
The Currency Converter window will open, allowing you to enter the amount, the currency you want to convert from, and the currency you want to convert to.
Click the Convert button to get the conversion result.
Click the Reset button to clear the input fields and start over.
Notes
The program uses the exchange rates API to get the latest exchange rates. If the API is down or unavailable, the program will not work.
The program will only work with currencies that are listed in the API. If you enter a currency that is not listed, the program will not work.
The program will only accept numerical values for the amount field. If you enter a non-numerical value, the program will not work.
The conversion result is rounded to 2 decimal places.
